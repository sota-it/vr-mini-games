# vr-mini-games

# VRミニゲームパック

Meta Quest 3に対応した、コントローラや手を使って直感的に遊べるVRミニゲームです。

## 開発環境
- Unity 6000.4.1f1 (Unity 6)
- Meta XR Core SDK
- XR Interaction Toolkit

## こだわった点・アピールポイント
- **操作性の向上**: XR Input Systemを用いて、VR特有の掴む・トリガーを引くといった操作を実装しました。
- **工夫**: 成功数の剰余を用いてレベルアップ演出を作成したほか、VRがはじめての人でもプレイできるようチュートリアルを作成しました。
- **種類豊富なミニゲーム**: ミニゲームを15種以上作成し、それらがランダムに登場するように構築することで何回でもプレイしたくなるように開発しました。
