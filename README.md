# VRミニゲームパック

Meta Quest 3に対応した、コントローラや手を使って直感的に遊べるVRミニゲームです。

[![ゲームデモ映像](https://github.com/user-attachments/assets/710729b3-2952-4a2e-8c74-8ed6a1fdb86d)](https://youtu.be/7mN8f1v2b6c?si=6QSw4GyFrDp82ZCW)


## 開発環境
- Unity 6000.4.1f1 (Unity 6)
- Meta XR Core SDK
- XR Interaction Toolkit

## こだわった点・アピールポイント
- **操作性の向上**: XR Input Systemを用いて、VR特有の掴む・トリガーを引くといった操作を実装しました。
- **工夫**: 成功した数に対して剰余を用いてレベルアップ演出を作成したほか、VRがはじめての人でもプレイできるようチュートリアルを作成しました。
- **種類豊富なミニゲーム**: ミニゲームを15種作成し、それらがランダムに登場するように構築することで何回でもプレイしたくなるように開発しました。
- **アニメーション**: UnityのTimeline機能を用いてゲーム開始時にアニメーションムービーを流しユーザー体験の向上を図りました。

## 同封したスクリプト
スクリプトはゲームの中心となる2つのスクリプトとミニゲームにて使用する1つのスクリプトをアップロードしました。
- **StartSceneManager.cs**: ゲーム起動時に開かれるSceneにて遷移のために使われるスクリプトです。
- **HomeManager.cs**: ミニゲーム終了後に成功・失敗の演出を表示し、次のミニゲームに遷移するために使われるスクリプトです。
- **FlyingGameManager.cs**: ミニゲームのうちの1つでコントローラの速度によって鳥が飛ぶような挙動を実現するために使われるスクリプトです。
