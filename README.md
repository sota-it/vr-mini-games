# vr-mini-games

# VRミニゲームパック

Meta Quest 3に対応した、コントローラや手を使って直感的に遊べるVRミニゲームです。

## 開発環境
- Unity 2022.3.x (LTS) ※ご自身のバージョン
- Meta XR Core SDK
- XR Interaction Toolkit

## こだわった点・アピールポイント
- **操作性の向上**: XR Input Systemを用いて、VR特有の掴む・トリガーを引くといった操作を違和感なく実装しました。
- **アルゴリズムの工夫**: 50近いスクリプト（.csファイル）を用いて、ミニゲームの勝敗判定ロジックや、テンポよくゲームが切り替わるシステムを自作しました。
- **特に見ていただきたいコード**: `Assets/Scripts/〜〜.cs` （※一番自信のあるファイル名をここに書く）
